# Copy as Markdown
A lightweight user script that converts web page content into clean Markdown format.

## Usage
1.  Navigate to any web page.
2.  Open your userscript manager's menu (usually by clicking the extension icon in your browser toolbar).
3.  Select one of the following commands:
    - Copy Selection as Markdown: Copies your currently highlighted text.
    - Copy Article as Markdown: Automatically finds the main article on the page and copies it.
5.  The Markdown content is now in your clipboard, ready to be pasted.
