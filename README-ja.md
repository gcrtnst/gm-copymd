# Markdown としてコピー
Web ページのコンテンツを、クリーンな Markdown 形式に変換してコピーする軽量なユーザースクリプトです。

## 使い方
1. 任意の Web ページを開きます。
2. ユーザースクリプトマネージャー（Tampermonkey など）のメニューを開きます（通常、ブラウザのツールバーにある拡張機能アイコンをクリックします）。
3. 以下のコマンドのいずれかを選択してください：
    - 選択範囲を Markdown としてコピー: 現在選択しているテキストをコピーします。
    - 記事全文を Markdown としてコピー: 記事本文を自動的に抽出してコピーします。
4. Markdown 形式のテキストがクリップボードにコピーされ、いつでも貼り付けられる状態になります。
